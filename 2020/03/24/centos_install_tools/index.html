4、查看网卡流量

界面说明:
“<=”与”=>”，表示的是流量的方向

“TX”：从网卡发出的流量
“RX”：网卡接收流量
“TOTAL”：网卡发送接收总流量
“cum”：iftop开始运行到当前时间点的总流量
“peak”：网卡流量峰值
“rates”：分别表示最近2s、10s、40s 的平均流量

可以通过键盘的”q”键退出iftop
