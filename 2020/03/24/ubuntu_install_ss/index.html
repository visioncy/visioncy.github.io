1、先备份一下原来的源镜像文件
cp /etc/apt-get.repos.d/CentOS-Base.repo /etc/apt-get.repos.d/CentOS-Base.repo.backup

2、下载新的CentOS-Base.repo，并写入到基础源文件中

#国内阿里的镜像

wget -O /etc/apt-get.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

3、apt-get缓存生成

apt-get clean all //清理apt-get缓存，使设置生效
apt-get makecache //将服务器上的软件包信息缓存到本地,以提高搜索安装软件的速度
apt-get -y update //更新本地apt-get

查看网卡流量
sudo iftop -i eth0

界面说明:
“<=”与”=>”，表示的是流量的方向

“TX”：从网卡发出的流量
“RX”：网卡接收流量
“TOTAL”：网卡发送接收总流量
“cum”：iftop开始运行到当前时间点的总流量
“peak”：网卡流量峰值
“rates”：分别表示最近2s、10s、40s 的平均流量

可以通过键盘的”q”键退出iftop

netstat -nat | grep -i “5222” | wc -l

cat /proc/cpuinfo| grep “processor”| wc -l
cat /proc/meminfo| grep “MemTotal”
